12 Changes to a schedule

12.1 Changes Order Details

When items are on a schedule it may be necessary to change the order details. In order to keep the customer satisfied, the user should be able to make these changes before removing the items off the schedule. 

These are the changes that could be made on a scheduled order: 
  • Stock Location;
  • Delivery Note Location;
  • Del Address; Required;
  • Time;
  • Immediate or scheduled;
  • Delivery Area, Damaged;
  • Colour Trim;
  • Qty;
  • Replacing items on a scheduled load for new items due to the customer changing their mind

Besides Qty and Replacement changes, the other changes are all made in “Change Order details” Section.

Quantity and replacement for new items are made in the Revise Account Screen. The reason for this is because the agreement total is adjusted.

If anychanges are made in “Change Order Details” screen the user will have to confirm these changes in the Changes after Scheduling screen.

12.2 Confirm Changes after Scheduling

The user will have to confirm any changes that have been made to a scheduled order in the Changes after scheduling. The item has been removed from the Load until this confirmation takes place.

Steps:

  • Search by Transport Picklist Number, Load Number or Revision Date
  • The original Delivery Note Details will appear in the top results grid. Click this line to populate “Original Item Details”
  • Click this line to see changes. The user can either confirm the change and re add the DN to the Load OR Remove it from the schedule completely. In which case, they will not be able to confirm the change in this screen again. The item will appear in the orders for delivery and collection screen.
  • If the Change is not confirmed or removed before the load Delivery Notes are printed for the rest of the load, the order will disappear from this screen but will be in the Orders for delivery and collection screen.

    • 12.3 - Revising Account - Changing Quantities

     An order should only be revised when the agreement total is being affected. Any other changes should be done in the Changes Order details screen.

    Steps:

  •  Remove the item or change the quantity of an existing item
  • The Schedule Override box will appear. Highlight the item to remove and “Delete”
  • Add a new item
  • Save and exit revise agreement
  •  Go to changes after scheduling
  •  The new item will appear in the New Item box. If only the quantity has changed the changes will appear in the “Changes” tab
  • Try and “Confirm Change or “Add it to a schedule” (Depends on if you changed the qty or added a new item. The following message will appear if the item hasn’t been DA’d yet:
  • “Item Must Be DA’d before adding to schedule” 
  •  DA the item. Re enter Changes after scheduling and “Add to Schedule” or “Confirm”.
