6.1 Immediate Delivery in Orders for Delivery and Collection

    Steps:
  1. Make sure the Pickup / immediate delivery Option is ticked
  2. Load. You will see that when the results are loaded the Delivery Note number is already visible. However, the delivery note must be printed before we can deliver the items fully.
  3. The Print Delivery Note button will be activated.
  4. Highlight the accounts to print delivery notes for by using the ctrl click button
  5. Print the Delivery Notes. A delivery note can only be printed once from this screen. After the note is printed this order will no longer appear in this screen.
  6. Go to section Immediate Delivery  to learn how to complete the immediate delivery process

 

6.2 Scheduled orders in Orders for Delivery and collection

    Steps:
  1. Make sure the Scheduling option is ticked
  2. Load. You will notice that the Print delivery note button is dimmed. This is because you will have to schedule the order onto a delivery load before printing the delivery note.
  3. From this screen the user has the option to print a pick list (If the country parameter is set to True). The pick list will contain all the items that were retrieved in the search and that are released.
  4. A pick list will be printed with a unique number. See Order Pick Lists section for more detail 

Additional Notes for this screen

Right click on as line to view customer comments recorded in the customer record such as directions to the house.
