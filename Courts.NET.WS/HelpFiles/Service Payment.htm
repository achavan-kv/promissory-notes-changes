Payment screen for Service

When a service request is not fully covered by a warranty, some or all of the cost of the repair can be charged to the customer. In this instance a new account will be created for the customer and related to the Service Request.

Payments will be credited to this account in the normal way using the existing Payment screen.

There is a SR Search button just below the current search button to call the Service Request Search Screen.This will allow the user to find the account by SR number or by customer id.

When the Payment screen loads any account that is related to a Service Request, the following title will be displayed in bold below the ‘Add To Potential’ field:

Service Request Number: 'nnnnnnnnnnn'

An indicator next to this field will show whether the payment is to an internal or external technician.
