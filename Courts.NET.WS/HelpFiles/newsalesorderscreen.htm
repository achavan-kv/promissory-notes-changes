New Sales Order Screen

 

Overview of the Delivery Process

 

The following is a brief overview of the current delivery process:

 

  1. Create the sales order and indicate per line item if it is for immediate delivery or scheduling. For collections do a Goods Return (go to step 5).

  2. Save and sanction as required.

  3. Take Payments as required.

  4. Authorise the delivery.

  5. Load the account in the Orders for Delivery & Collection Screen.

 

For Scheduled Orders:

 

  1. Print Pick lists.

  2. Schedule the delivery.

  3. Print the Delivery / Collection note and Delivery / Collection Schedule.

  4. Notify the delivery / Collection.

 

For Immediate delivery:

 

  1. Print the Delivery notes.

  2. Immediately deliver / collect the items.

 

Delivery Type (Immediate or scheduled)

 

You are now able to select whether the item you are adding to the sale is to be for Immediate Delivery or to be scheduled for delivery. This should be done for each line item added to the sale.

 

  • The ‘Immediate’ tick box should be ticked for Immediate deliveries.

  • If the ‘Immediate’ tick box is left un-ticked, then it will be assumed that the item will be ‘scheduled’ for delivery.

  • Subsequent items selected should remember what the initial item’s selection was for and default to that.

 

Delivery Area

 

There is a drop down called ‘Delivery Area’. This is if the ‘Customer’ would like the goods on the order to be delivered to different parts of the country.

 

  • Please note that delivery area’s should be set up from the ‘Delivery Area Maintenance’ screen.

  • The user should select the relevant delivery area per item.

  • Subsequent items delivery areas should default to the first selected delivery area.

  • If the 'Delivery Required' date is set to a date not in line with the day of the week that the delivery area is set for then a pop up should appear displaying the days that are available.

  • You should be able to continue even if the incorrect day is selected or have the option to cancel and select a new day.
