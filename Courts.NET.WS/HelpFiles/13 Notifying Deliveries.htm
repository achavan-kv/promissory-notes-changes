13 Notifying deliveries

This is the process for notifying a delivery:

  1. Enter the branch number
  2. Enter the load date. This can be any date in the past. If the date is more than 3 weeks ago a message will appear asking the user if they would like to continue.
  3. Enter a load Number. This would have to be a load number where delivery notes have already been printed for that load.
  4. Search
  5. The delivery notes on the schedule will appear
  6. To notify the deliveries on that schedule, click Notify deliveries. This will notify all delivery notes that were on that load.
    7. If certain items were not delivered and needed to be removed from the load then the following process should be followed
    1. Highlight the delivery note in the ‘delivery notes on schedule’ section
    2. The line items on that delivery note will appear in the ‘Line items on delivery note’ section
    3. The section ‘Stock items for account’ lists all the stock items for the account selected in the ‘Line items for delivery note’ section.
    4. Highlight the line item to remove
    5. Click ‘remove line item’
    6. A reason has to be captured for removing an item from a delivery note
    7. Reasons for removing items from a delivery note need to be set up in codes maintenance under the code DS1.
    8. This line item will be removed from the schedule and will go back to a status of waiting scheduling. The delivery note number will remain the same. There will be an undelivered indicator added to that line item ‘A’. Each time this item fails delivery the undelivered indicator will increment through letters of the alphabet i.e. B, C, and D etc.
    9. If the user needed to remove the entire contents of a delivery note, they would highlight the delivery note and press ‘Remove delivery note’.
    10. In certain cases where multiple quantise of the same item are on the load some of them may be delivered but others not. In this case the user will have to change the quantity delivered.
    11. This can be done by highlighting the line item and pressing the ‘Change Quantity’ button. Quantities can only be reduced. The final remaining quantity of an item cannot be less than one.
