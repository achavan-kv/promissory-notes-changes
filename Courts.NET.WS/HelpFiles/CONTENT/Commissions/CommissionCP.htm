Commission Country Parameters

  • Maintenance screens to be available under System Configuration, however need product level (sku) on the SPIFF screen to be an additional user access right.
  • Additional access right for Supervisors to be able to add display item SPIFF onto NSO. This is because the system won’t know if it is a display item being sold as FACT2000 does not hold display stock separately.
  • Country parameter –SPIFF Reselection To highlight in NSO screen that there is a similar product to the one entered in NSO with a higher SPIFF.
  • Country Parameter – Months to retain Commission – Number of months to retain commission records.
  • Country parameter – to determine whether popup enabled for alternative SPIFF suggestions
  • A CSR should only be able to enquire to look at commissioned earned for them. Managers can view all staff. Additional Right – Commission Enquiry Screen – View Employees
  • Country Parameter – Maximum Sales Commission rate default 50%
  • Country Parameter – Maximum SPIFF Commission Amount default 999,999
  • Lose Collection \ Cancellation commission - This a yes or no. If yes the CSR will lose commission on cancellations
  • Lose Rebate Commission - This is a yes or a no. A yes will mean that commission earned on the rebate amount will be deducted if the account is settled early.
  • Lose Repossession\Cancel Commission Months - The amount of month within which if an item is repossessed the CSR will lose commission earned.
