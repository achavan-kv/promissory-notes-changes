Commissions in the New Sales Order Screen

Extra spiff for display items

There will be a right click option on an undelivered product (stock) line called Add additional SPIFF. When choosing this the user will be presented with a value to enter and a user name and password for authorisation

  • The system will allow authorisation based on the user right “New Sales Order – add extra spiff”
  • The system will record the user name who authorised, the amount of spiff, account number, item number and stock location in the SalesCommissionExtraSpiffs table. The system will prevent more than one SPIFF being added against an item.
  • When the item is delivered the extra commission will be delivered in the commission process
  • The extra spiff record could be removed by right clicking on the SPIFF line and choosing remove spiff – this will only be possible until the item is delivered. Clicking on a SPIFF line will not populate the Product information line. This must not be printed on the agreement.

Spiff Selection

Depending on the country parameter, SPIFF Reselection , the Spiff Selection pop-up will behave as follows:

  • Spiff Reselection set to PS – A pop up screen will be displayed listing items in the same category as the entered item, that have a higher SPIFF value and that product is in stock. The entered item must have a SPIFF against it.
  • SPIFF Reselection set to PA - A popup screen will be displayed listing items in the same category as the entered item, that have SPIFFs’ against them, even if the entered items does not have a SPIFF against it.

This functionality will appear before the item is clicked on and entered into the product grid and so will appear before any warranties are added or the out of stock pop up and associated items. If associated item selected has a SPIFF the SPIFF popup may need to be invoked again.

If a SPIFF’d item has linked items in stock (available stock) then a Gold dollar sign will appear. If this is clicked the multiple SPIFF items will be displayed, together with the item description, SPIFF value, and SPIFF percentage.
