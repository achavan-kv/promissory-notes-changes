Commissions Report

To List out totals for users within a date period which will be based on the delivery notification date.

There will be 2 checkboxes both initially checked: Standard Commmision, SPIFF. This will display the following information in the header record:

  • Employee No + Name,
  • Commission Total,
  • Rebate Total,
  • Repossession Total,
  • Cancellation Total
  • SPIFF total,
  • Product total,
  • SC% delivery amount

 

On clicking on a line for an employee will show the detail record for that employee for that date period:

  • Run date
  • Account number
  • Invoice No (for cash and go)
  • Commission Type – with description not just the 2 digit character
  • Item Number
  • Branch
  • Commission Amount
  • Percentage (Commission Amount/Delivery Amount),
  • delivery amount

 

Both sets of visible data extract should be able to be exported into Excel.
