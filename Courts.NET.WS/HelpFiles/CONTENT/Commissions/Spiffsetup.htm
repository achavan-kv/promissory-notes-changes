Setting up Spiffs

  • The Spiff Setup can be found in the SPIFF tab of the commission set up screen. The spiff set up screen works in the same way as the commissions set up. The only differences are that the drop down consists of SPIFF and Linked Spiff rather than product codes and level.
  • A SPIFF is a once off amount allocated to specific product codes. This is in addition to any other commission earned on the product.
  • A SPIFF can be a value or a %.
  • A linked SPIFF is an incentive that encourages users to sell items together. When Linked spiff option is chosen you will see 4 more fields appear at the bottom of the screen. The intention is for the user to set up linked items that when sold together will earn the saleperson a percentage or value of the sale. For example:
  • A TV and a HIFI are set up as linked SPIFFS with a 10% SPIFF. If the user sells the HIFI and TV together they will earn 10% on the TV and 10% on the HIFI.
  • If the user only sells the TV, they will earn nothing. UNLESS the TV was also set up as a Single Spiff.
  • A linked SPIFF overwrites a single spiff.
  • A SPIFF can only be earned on a Stock Product not a non stock. The system will not allow you to enter a non stock item here.
