Service Progress Report

This report will list Service Requests that are outstanding, allocated or complete. A date range will be mandatory. Three tick boxes will include SRs that are outstanding, allocated or complete. These tick boxes will default to ticked. The report can optionally be restricted by a product code or by a product group. If the user enters a product code then the product group will be set to the corresponding group and become read only.

The report results will be generated in a Data Grid View when the search button is pressed. This will enable an excel button to allow the results to be copied to file and printed.

The report columns will be:
  • Service Request Number
  • Date Logged
  • Logged By
  • Debtor Account No
  • Service Type ~ (C)ourts Account / (N)on Courts / (I)nternal
  • Customer Name (combine first and last)
  • Customer telephone no's: home, work, mobile
  • Arrears
  • Purchase Date
  • Product Code
  • Product Description
  • Model Number
  • Serial Number
  • Warranty Y/N
  • Contract Number
  • Resolved using soft scripts Y/N
  • Deposit Paid
  • Repair Estimate
  • Technician
  • Repair Date
  • Parts Expected Date
  • Part Number ~ NOTE: for multiple parts the report line will repeat to list all parts
  • Part Quantity
  • Part Description
  • Resolution Code
  • Resolution Description
  • Resolution Date
  • Replacement Y/N
  • Food Loss Y/N
  • Primary Charge To
  • Parts Cost
  • Labour Cost
  • Additional Labour Cost
  • Total Cost
