Service Request Audit

This screen is accessed from other screens that pass the SR number. This screen will not contain a search option, only the current SR, Customer ID, and type of Service Request. These are all read only fields.

 

The Audit information is headed by the following dates. These dates show the latest instance of each event:

  • Date Logged - Date the SR was logged.
  • Date Allocated - Date the SR was allocated to a technician
  • Date of Deposit - Date the deposit is paid.
  • Date of Payment - Date the payment for the outstanding charges is paid.
  • Date Resolved - The date the repair is resolved.
  • Date Re-Opened - The date when the reopen option is chosen and comments are made against an account

The Audit information in the "Updates" list will show all the instances of each event. There will be a date and the employee number next to each event as well as a description providing more specific information on the event.

Possible events and their descriptions are:

 

Event Description
Open Employee Number
Allocation Name of Technician and the repair estimate if entered
Pay Deposit Amount and the debtors account
Resolve Resolution Description and the cost of the claim
Payment Amount and the debtors account number
Closed Account Number
