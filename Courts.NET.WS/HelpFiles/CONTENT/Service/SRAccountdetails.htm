Service Request Summary on Account Details

There is a tab in the Account Details screen called ‘Service Summary’.

The background colour of this tab will be pale red if there is an open SR.

Products related to an SR will be listed using the columns:

  • Service Request Number
  • Date Logged
  • Product Code
  • Product Description
  • Resolution Date
  • Replacement Status - ‘A’ = Authorised; ‘C’ = Collection notified; ‘D’ = New Item Delivered
  • Deliverer name

The user will be able to right click a row in this list to open the Service Request screen with this Service Request selected

The Telephone Caller and DCA print outs will include the Service Summary details.
