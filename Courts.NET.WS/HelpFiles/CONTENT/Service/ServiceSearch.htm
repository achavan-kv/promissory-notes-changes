Service Request Search

This screen will allow the user to search either by a Service Request Number or by a customer ID.

The results can be filtered by the different types of SR using the tick box options:

  • Courts Account
  • Non Courts
  • Stock Repair

The results can list more than one SR when searching by Customer Id. The result columns will be:

  • SR number
  • Date SR Logged
  • Logged by
  • Status ~ ‘N’ = New/re-opened; ‘E’ = estimate; ‘A’ = allocated; ‘R’ = resolved; ‘C’ = closed; 'D' = deposit required
  • Product Code
  • Product Description
  • Delivery Note Number
  • Delivery Date
  • Extended Warranty
  • Contract numbers

The user will be able to right click a line in the results for the following options:

  • Audit Details - navigate to the Service Request Audit screen passing this SR
  • Customer Interaction Details - navigate to the Customer Interaction screen passing this SR
  • Service Request - navigate to the Service Request screen passing this SR
