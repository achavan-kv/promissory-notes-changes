Outstanding Service Requests

This screen will provide a set of reports for open Service Requests. A drop down will allow the user to select a type of report. A date range and/or an SR number range may be optionally entered to filter the results. The ‘Not Updated Since’ and ‘Technician’ fields will only be enabled for certain report types.

 

The report results will be generated in a DataGridView when the search button is pressed.

This will enable an excel button to allow the results to be copied to file and printed.

The set of reports are:

  • Unallocated
  • Awaiting Deposit ~ only included if the country parameter ‘SERVICE Deposit’ is enabled.
  • Not Updated Since
  • By Technician
  • By Date Logged
  • Awaiting Spare Parts
  • Awaiting Payment
  • Awaiting Estimate

Unallocated

These are all SR's not yet allocated

The report columns will be:

  • Service Request Number
  • Date Logged
  • Deposit paid Y/N - only visible if the country parameter ‘SERVICE Deposit’ is enabled.

Awaiting Deposit

These are all accounts awaiting deposit

The report columns will be:

  • Service Request Number
  • Date Logged
  • Deposit Amount

The user will be able to right click a row to open the standard authorisation prompt. When authorised the deposit amount will be reset to zero and the deposit paid indicator set to ‘Y’.

 

Not Updated Since

All SR's that have not been updated within the specific date range. This report will pick up when thetre are no changes with these dates:

  • Date Logged - ‘SR_ServiceRequest.DateLogged’
  • Date Allocated - ‘SR_Allocation.DateAllocated’
  • Date Re-opened - ‘SR_ServiceRequest.DateReopened’

The report columns will be:

  • Service Request Number
  • Date Logged
  • Technician
  • Date Allocated
  • Repair date

By Technician

This is a report of all SR's allocated to a technician

The report columns will be:

  • Service Request Number
  • Date Logged
  • Technician
  • Repair date

 

By Date Logged

This will bring back all SR's by date logged that are not yet closed or resolved.

The report columns will be:

  • Service Request Number
  • Date Logged
  • Technician
  • Date Allocated
  • Repair date

 

Awaiting Spare Parts

All SR's awaiting spare parts

The report columns will be:

  • Service Request Number
  • Date Logged
  • Technician
  • Date Allocated
  • Parts Expected Date
  • Repair date
  • Product Code
  • Product Description

 

Awaiting Payment

All accounts awaiting payment.

The report columns will be:

  • Service Request Number
  • Date Logged
  • Date Allocated
  • Repair Date
  • Outstanding Balance

 

Awaiting Estimate

All accounts Beyond economic repair that are awaiting an estimate.

The report columns will be:

  • Service Request Number
  • Date Logged
  • Product Code
  • Product Description #
