18 Cancel Collection Note

Sometimes the customer might request a collection for various reasons but then change.

Use this screen to cancel planned collections between the stages of doing the goods return until the collection note is printed.

Notified Collections cannot have the collection note cancelled.

 

  • Enter the account number
  • Press delete to reverse the cancellation
  • The account will appear as if the item was never returned.
  • An audit trail of the attempted collection and the cancelled collection will be tracked in the 'Transactions' tab of account details.
