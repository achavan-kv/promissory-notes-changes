10 Delivery scheduling

This is where delivery loads are set up.

  1. The 'transport available' grid shows all the available trucks. These are set up in transport maintenance.
  2. Highlight a truck, assign a load number and press add. The load will appear in scheduled loads
  3. The next step is to add Pick lists and / or delivery notes. There are country parameters controlling whether a country can
  4. Add only one pick list per load
  5. Add multiple pick lists per load
  6. Add delivery notes not already on a pick list to schedule
  7. Once the orders are added to the schedule press the delivery details screen.
  8. From this screen the delivery schedule and delivery notes are printed
