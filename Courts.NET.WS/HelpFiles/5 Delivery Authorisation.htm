5 Delivery Authorisation

Search for accounts that need to be Delivery Authorised using the appropriate search parameters.

  • Include Cash
  • Include HP
  • Include RF
  • Exclude Accounts with no products
  • Deposit or Instalment paid / not paid

    •  

    To clear a flag on the account, right click the flag and click clear.

    Flag descriptions are user maintainable in codes maintenance under code PH2 – Proposal Hold Flags.

    Examples of Hold Flags are:

    • Sanction Stages not complete
    • Customer Details need to be verified

     

    There is a Summary Tab in Authorise delivery which assists users when clearing the account for DA. This screen includes the following tabs with the following fields:

    Personal / System Recommendation Tab
    • System Recommendation – This says if the account has been accepted or referred
    • Age
    • Marital status
    • Residential Status
    • Occupation
    • Expense
    • Income after Tax
    Agreement Details Tab
    • % repayment to disposable income
    • Account Type
    • Terms Type
    • Deposit
    • Instalment
    • Agreement Total
    • Credit Limit
    • Number of RF Accounts
    • Line Items
    Account History
    • Arrears
    • Outstanding balance
    • Largest Agreement
    • Fees and interest since account last opened
    • Total Instalment value of current
    • Number of current, settled and account in arrears
    • Number of returned Cheques
    • Worst status ever
    • Longest agreement
