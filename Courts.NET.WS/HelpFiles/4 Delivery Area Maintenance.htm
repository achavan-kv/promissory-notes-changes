4 Delivery Area Maintenance

4.1 Amend / View and Existing delivery Area

 
  1. Enter Delivery Area Maintenance from the System Configuration menu
  2. Choose the area from the dropdown
  3. The Selected time slots for that area will appear.
  4. The user can add or remove slots as required using the arrow buttons in the middle of the screen.
  5. The user is also able to choose the branches these areas are applicable to.

4.2 Creating a new Delivery Area

 
  1. Go into Delivery Area Maintenance
  2. The Name and Description field will be dimmed
  3. Add time slots to activated these fields
  4. Enter a name for the new area
  5. Enter a description
  6. Save and Exit.
