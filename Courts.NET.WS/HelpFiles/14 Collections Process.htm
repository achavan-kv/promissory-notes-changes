14 Collections Process

 

The process is:
  1. Enter the Goods Return in the Goods return screen
  2. Choose one of the following:
    • Repossession
    • Cancellation
    • Exchange
    • Identical Replacement
  3. Select the item to be returned
  4. The return of warranties is discussed in the Warranties chapter
  5. Choose the reason for collection
  6. Identify if the collection is Immediate or for scheduling
  7. Process the return
  8. Load the order in the Orders for Delivery / Collection screen
  9. For Immediate Collections:
    • Print the collection note for immediate deliveries
    • Record the immediate delivery
  10. For Scheduled Collections:
    • Print the pick list for scheduled items or print the excel spreadsheet to get the collection note numbers to enter in the Delivery Scheduling screen.
    • Schedule the collection
    • Print the Collection notes and collection schedule
    • Notify the collection
