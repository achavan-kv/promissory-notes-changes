Status Code Changes

This screen is used to change the status of an account. Valid status codes for an account are as follows:

Status Code Description

0 -Missing Details

U -Unpaid deposit prior to delivery

S -Settled – balance must be 0 before being allowed to settle.

1 -Account is up to date

2 -Account is 1 instalment in arrears (or 90% of 1 instalment in arrears to prevent the customers just paying a little amount)

3 -Account is 2 instalments in arrears

4 -Account is 2 instalments in arrears and has made no payment since delivery or account is 3 instalments in arrears

5 -Account is 3 or 4 instalments in arrears

6 -Account is marked for write off/repossession

7,8- Legal Action

9 -Staff Account

Any changes to status codes can be seen from the account details screen, as well as this screen.  The system will record the employee who changed status code and the date changed.
