6 Orders for Delivery and Collection

After Delivery Authorisation the account will be awaiting Scheduling or Immediate delivery. It will appear in the orders for Delivery / Collection screen.

Search parameters:

Branch / Warehouse

  • This is the branch where the delivery of the goods will take place from.

Del Note Branch

  • This is the Delivery Note Branch for the items. Delivery Note Branch isn’t always the same as the stock location

Delivery Area

  • If you want to load results by the delivery area

Product category

  • There are 3 basic categories (IT, Furniture and Electrical). The second drop down allows the user to further breakdown the chosen category I.e. Electrical into vision, audio, major whites etc

Order dates

  • Select the date range from when the order was taken

Required Delivery Date

  • This is the date delivery is required

Delivery Process

  • Choose if the list should load for immediate delivery or scheduled items

Include

  • Limit the load to either Deliveries or collections.

Linked Items

  • When searching by category there may be other items on the same order not in the same category. If the user wants to display these items they should tick this box

Assembly

  • This should be ticked if searching for items that need assembly.

 

There is an option to export the list to Excel by pressing the Excel Icon.
