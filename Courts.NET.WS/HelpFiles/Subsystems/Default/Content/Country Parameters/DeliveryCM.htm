1 Delivery Country Maintenance

The following country parameters can be used to customise the deliveries and collections process to each business.

1.1.1 Allow DA for unpaid accounts
  • This parameter controls whether the users will be allowed to DA accounts which are unpaid.

1.1.2 Allow Delivery Note to be added to Load 
  • This allows delivery note details to be added to trucks in the delivery scheduling screen.

1.1.3 Default Delivery Days
  • Determines the default required delivery date on the new account screen for new accounts

1.1.4 Default Delivery Note Branch
  • This is generally the branch number of the warehouse for the region but in some cases may be different. Singapore uses a central branch to print all delivery notes for the various warehouses.

1.1.5 Delivery percent for date first
  • An Item will be considered delivered when this % of the goods has been delivered

1.1.6 Display Assembly 
  • To determine whether the assembly field is displayed on New Account/Revise Agreement, Orders for Delivery screen and the Transport Picklist screen.

1.1.7 Enable Delivery Note Branch
  • If enabled the delivery note branch should be displayed in the New Sales Order, Orders for Delivery, Amend Picklist and Transport Picklist screens. If this is enabled then the Stock Location dropdown will be disabled.

1.1.8 Fixed Date of first Instalment 
  • Determines whether Date of first instalment is fixed
    • 0 for not fixed,
    • 1 for fixed and
    • for fixed due day but date of first instalment will always be a set number of days after delivery

1.1.9 Number of Transport Slots 
  • This is number of transport slots in a day. I.E. am, pm or 9-12, 12-3 etc

1.1.10  One Pick list per truck 
  • Limit the number of pick lists that can be added to a trucks load to one.

1.1.11 Orders for Delivery - Print Pick Lists
  • If enabled the system will display the print pick list button in this screen.

1.1.12 Orders for Delivery - Required Delivery Date 
  • This option will determine whether the order date or required delivery date will be used as a search parameter in the Orders for Delivery and Transport Pick List screens. If true it will be based on the required delivery date.

1.1.13 Restrict Repossessions by Account Status 
  • This parameter restricts repossessions to accounts in status code 3, 4, 5 or 6 only
