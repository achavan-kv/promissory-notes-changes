1 Warranty Country Parameters

1.1.1 Warranty Expiry Prompt

Number of days prior to a warranty expiring that the pop-up for an account will appear.

 

1.1.2 Allow Automatic Return Codes For Warranties

This parameter allows return codes to be automatically generated for warranties being collected.

 

1.1.3 Credit Warranties days before payment due

Minimum days before payment becomes due on credit warranties

 

1.1.4 Credit Warranties Number of days grace

If the customer hasn't paid for his warranty on credit in the allotted days he will this many days grace until the warranty is terminated by the system

 

1.1.5 Credit Warranties Days before reminder letter

The system will send out a reminder letter to the customer to pay for his warranty on credit this many days after the purchase

 

1.1.6 Credit Warranties return Item Number

When the system automatically returns a warranty because the customer has failed to pay then this is the return item number which will be sent to FACT2000

 

1.1.7 Instant Replacement Period 1 (days)

The number of DAYS that the manufacture warranty covers an item for Instant Replacement

1.1.8 Instant Replacement Period 2 (months)

The number of MONTHS in addition to the manufacture warranty where the warranty remains in place and Courts will pay the cost of replacement

 

1.1.9 Instant Replacement Period 3 (months)

The number of MONTHS in addition to period TWO where the warranty terminates and Courts bear the cost of replacement. After the end of this period AIG will pay the cost of replacement.

 

1.1.10 Automatic Contract Number Entry

Uncheck this box if the user has to enter the Warranty Number Manually otherwise the system will generate a contract number

 

1.1.11 .NET Automated Warranty Contract Print

Tick here if the system is printing Automated Contacts

 

1.1.12 Allow Instant Replacement On Credit

Determines whether warranties can be purchased on credit for instant replacement items when doing a cash and go sale. This might be because the customer has only bought in enough cash for the item but would also like the warranty. In this case they are allowed to purchase the warranty on credit.

 

1.1.13 Warranty Cancellation Days

This is the period of time in days from delivery that a warranty can be cancelled in isolation to a product.

 

1.1.14 Number of Warranty Contract copies

Number of copies of the warranty

 

1.1.15 Customer Details for Cash & Go Warranty

Allows entry of customer details on a Cash & Go account with a warranty

 

1.1.15 Warranty after delivery days

Maximum number of days after delivery that a warranty may be added

 

1.1.16 Warranty Expiry letter generation in months

Number of months prior to a warranty expiring that a letter will be sent to the customer.

 

1.1.16 Maximum prompt days after warranty expired

Maximum number of days to prompt for renewal after the warranty has expired

 

1.1.17 Warranty Expiry Settlement prompt in Days

Number of days prior to a warranty expiring that the pop-up for an early settlement to appear.

 

1.1.18 Warranty Contract Style sheet

Type in the name of which style sheet to use for warranties. It will either be using Preprinted format or Plain paper format

 

1.1.19 Warranty Validity Months

The number of months that a warranty will remain valid on a replacement or exchanged product. This means that the new item added to the account will be linked to the original warranty.
