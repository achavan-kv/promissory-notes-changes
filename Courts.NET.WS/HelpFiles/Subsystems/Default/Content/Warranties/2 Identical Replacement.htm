2 Identical Replacements

Identical Replacements are not very common but can occur when the delivered item is damaged and the item needs to be replaced with another identical item.

 

  1. When a replacement is processed in the Goods return screen, the delivery note for the new item is automatically generated at the same time the collection note is generated. This is because logistically the warehouse may want to deliver the new item and then pick up the old item in one load rather than doing the collection and the processing the new delivery.
  2. When the identical replacement option is chosen in the goods return screen CoSACS will check if the warranty attached to that product is either within its warranty validity period or not.
  3. If the warranty is still within the validity period then this warranty is automatically attached to the replacement item.
  4. This warranty will not be collected when the original item is returned in the goods return screen. An exchange and identical replacement is different to a repossession or a cancellation because the warrant is being used as it was intended.
  5. If the warranty is not within the warranty validity period then the warranty will not be reattached to the replacement item as it is deemed to be fulfilled and the item no longer under warranty cover. If the customer would like a new warranty then a warranty can be sold to the customer.The original warranty remains on the account but is not linked to any stock items
  6. For a replacement or exchange the warranty is not actually being returned. It is been used and remains on the account. When identical replacement is chosen in the goods return screen the Warranty Item will disappear. This is to prevent the user from trying to return a warranty while processing a replacement or exchange.
