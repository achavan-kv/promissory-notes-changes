8 Warranty Reports

Warranty reports can be accessed from the Reports menu.

Reports can by grouped by one of the following:

  • Account Type
  • Category
  • Branch
  • Sales Person

The report can also be filtered according to user defined categories. This is done by using the Category Set Maintenance screen. This is accessed from the category drop down.

The report can also be based on either Warranty Delivery date, Item Delivery date or Order date for various reports. Some reports are restricted to a specific date type.

Here is a list and description of each of the reports:

Supashield Sales

This reports on all Supashield Sales based on the date range chosen. It does not matter if the SS has since been cancelled. It will still show. Cancellations will show on the cancelled warranties report.

Instant Replacement Sales

A report of all IR sales.

Instant Replacement Claims

A report of all IR Claims in the period chosen.

Warranties due for renewal

This will report on all warranties that have expired or due to expire in the period chosen. Each row in the result will have a status of either: Due to expire, Expired and expired - passed renewal period.

All warranties passed renewal period will not be renewable. All the other warranties displayed will be renewable.

Renewal Sales

This is a report on all Renewal Sales. This report is based on Delivery date

Second Effort Solicitation

This is a report on warranty sales which occured after the original item was delivered.

Warranty Repossessions

A report on Warranty repossessions

Warranty cancellations

A report on Warranty Cancellations

Missed Warranty Sales

This is a report on all Warrantable Sales where a warranty was not sold.

Warranty Hit Rate

This reports on the % of Warrantable Sales which where actually sold with a warranty.

Insurance Claims on warranties

This will report on all Insurance claims which used the code INW. In the general financial transactions screen, when a user processes an INW on an account they will be prompted to choose the warrantable product being claimed against (if there is one). Courts are due a refund in these circumstances. The other reports won’t pick this up as it is not recognised as a Cancellation or a Repossession

Lost Warranty Sales Commission

This is a report on all the Missed Warranty Sale opportunities and how these misses affect the commission of the sales person.

A country parameter will be required for this report. It is called Target Hit rate %.

Currently there is a target hit rate % assigned to warranty sales. For example:

Total Warrantable Sales Value= 1000

Target hit Rate % = 40%

Therefore required sales value for the sales person need to be 400 or above (1000 * 40%)

The Lost Sales Value is calculated as:

(Total Warranty Sales Value – Required Sales value. Only negative results should show here. If the value is positive it means that there was no lost sales value and the report should show a zero value)

Lost units is calculated as:

(Actual Warranties Sold Qty - (Total Warrantable Sales Qty * Target Hit rate %)). Only negative results should show here. If the value is positive it means that there was no lost sales units and the report should show a zero value

Lost Commission is calculated as:

(Lost Sales value * Commission rate against that warranty)
