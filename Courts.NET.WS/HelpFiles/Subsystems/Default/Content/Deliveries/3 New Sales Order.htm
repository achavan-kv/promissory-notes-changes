3 New Sales Order Screen (Delivery fields)

  • For each line item the user will have to select if this item is for immediate delivery or for scheduling.
  • Tick the immediate box for immediate delivery. Or leave blank for scheduling.
  • A delivery area can now be chosen per line item on the order. This is if the Customer would like the goods on the order to be delivered to
  • The Delivery Note Branch will be dependant on codes maintenance set up.
    • The Delivery Note Branch will be dependant on codes maintenance set up.
    • A default delivery note branch can be set for all products in country maintenance. If set to zero the Delivery note branch will always be the same as the stock location.
    • However it is also possible to have default delivery note branches per product category.
    • In codes maintenance the REFERENCE field for each product category can be set to the default delivery note branch for that category.
    • The code categories in codes maintenance are:
      • PCE - Electricals
      • PCF - Furniture
