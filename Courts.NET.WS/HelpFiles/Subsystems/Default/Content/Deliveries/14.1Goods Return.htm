14.1 Goods Return

Country parameters associated with this screen:

“Restrict repossession to status codes”.

Goods return Process:
  1. Enter the account number
  2. Search
  3. If the repossession drop down is enabled choose if this is a Return or a Repossession
  4. If its repossession choose the type of repossession i.e. voluntary, part or full.
  5. Repossessions to not require any further collection scheduling. They will always show as a to follow on the account unless the account is redelivered after repossession.
  6. If it’s a return, choose whether it’s a credit or a replacement
  7. Choose the reason for the return. Reasons for return are populated by the code “ RGR- Reasons Goods returned” in codes maintenance
  8. Choose the delivery area the item is to be collected from.
  9. Select whether the return is for Scheduled or immediate collection
  10. Highlight the items to return.
  11. If there is a warranty associated with the returned product the user will see a prompt reminding them that a warranty exists on the account.
  12. Return them by pressing the “+” button.
  13. Save and exit
