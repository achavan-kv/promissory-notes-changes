11 Transport Pick Lists

The Transport Pick List looks very similar to the Order Picklist screen. The difference is that this screen will only retrieve orders on a load where the Delivery note has not yet been printed.

The user has an additional search parameter “Truck ID”.

There is no option to Print Delivery Notes from this screen.

The Pick List can be printed in 2 ways

  • Print Transport Pick List Button
  • Export the Pick List to excel

Both methods will generate a transport pick list number.
