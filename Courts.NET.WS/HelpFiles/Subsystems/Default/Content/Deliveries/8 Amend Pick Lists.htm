8 Amend / Reprint Pick list

Order Pick lists or Transport Pick lists can be amended or reprinted from this screen. The screen is located off the warehouse menu.

Click here to see the steps:
  1. Choose the correct branch where the order pick list or transport pick list was generated
  2. In the “PICK LIST Number” field, select a pick list number from the pick list drop down. The only pick list numbers displayed here will be pick lists that have not been part of a load already notified
  3. If the pick list is already on a scheduled load then you cannot remove or add items to the pick list. You can only reprint the pick list from this screen
  4. If the pick list has been printed but has not been added to a load then the user will be able to remove or add items from this pick list
  5. The user can also remove an entire pick list from this screen.
  6. Load (Choose the load button that is aligned with the pick list drop down).
  7. The Pick list details will be populated
  8. In order to add more delivery notes to this pick list the user will have to enter the Delivery note number.
  9. Load
  10. The delivery note will appear in the order details
  11. Choose the pick list from the drop down
  12. Load.
  13. Highlight the Delivery note to add to the pick list
  14. Click the add item to pick list button. The item will be added to the pick list.
  15. Use the print additions button to print the additional items added to the pick list
  16. In the “Transport Pick List Number” field, select a pick list number from the pick list drop down. The only pick list numbers displayed here will be pick lists that ARE on a load NOT yet notified or had the delivery note printed.
  17. The user will only be able to Reprint the Pick list or Print Additions. They cannot add any items to the Pick List in this screen.
  18. The assumption is that all items on a LOAD come from the same stock location.
