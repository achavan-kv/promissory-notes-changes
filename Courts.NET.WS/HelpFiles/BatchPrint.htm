Batch Printing

  • This screen can be accessed from Service --> Batch Print.
  • The user will be able to specify an SR number range, a date range and a technician.
  • At least one of these criteria must be entered before searching.
  • The search button will list all matching SR's.
  • The 'Print Selected' button will only be enabled if one or more lines are selected.
  • 'Print All' prints all requests retrieved from the search.
  • The Service Request as well as the soft script answers will be printed.
